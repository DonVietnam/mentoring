## Разделение доступа к Google календарям
При создании календаря сделать его общедоступным и открыть доступ по ссылке, затем скопировать id календаря.

Для управления доступом к каледарям нужно модифицировать файл _config/shared-calendars.yml_
```
batch-1: &batch1 bbp2fbora4vteeuiqp45l26qog@group.calendar.google.com

team-1:
  id: 1
  common:
    - *batch1
  personal:
    - kup8op97jncic71nt8t4t38p90@group.calendar.google.com

team-2:
  id: 2
  common:
    - *batch1
  personal:
    - 8q9n46iqrd6eflvjcrc8i85va8@group.calendar.google.com
```
В начале файла указываются id общих календарей для каждого батча. 
При создании нового батча указать: имя батча, ссылку на батч и id календаря, связанного с этим батчем.

Ниже нужно описать каждую команду по шаблону:
- id - идентификатор команды 
- common - список общих календарей
- personal - список персональных календарей

После изменений запушить файл в репозиторий.

## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
